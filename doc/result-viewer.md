# Result Viewer

The Result Viewer is a HTML-based tool for analyzing and visualizing benchmark results. It provides:

- Interactive result exploration and filtering
- Performance metric visualization
- Export capabilities for further sharing and analysis

You can find the Result Viewer in the `build/` directory after building the project.

Import the `result.csv` and `analysis.json` files generated by the CLIPBench CLI application. Use the viewer to visualize the performance evaluation in 1D or 2D plots. The plots are interactive, you can use zoom, selection of part of space for better exploration and you can inspect specific points by hovering over them. The choice of variables to be plotted in multi-dimensional cases can be advised by Permutation Importance analysis, which is part of the `analysis.json` output and also visualized in the viewer. The analysis includes performance metrics such as minima, maxima, mean or outliers.
